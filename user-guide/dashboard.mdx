## Overview

The Dashboard is your central hub in MediaStream, providing quick access to your media content and key metrics. After logging in, you'll land on the Dashboard which displays an overview of your streaming platform.

## Accessing the Dashboard

The Dashboard is accessible from multiple locations:

- **After Login**: You're automatically redirected to the Dashboard after successful authentication
- **Sidebar Navigation**: Click on the **Dashboard** icon (grid icon) in the left sidebar at any time
- **Logo Click**: Click on the MediaStream logo in the sidebar header to return to the Dashboard

## Dashboard Layout

The Dashboard features a clean, organized layout with the following components:

### Top Navigation

- **Breadcrumbs**: Shows your current location ("Dashboard")
- **User Menu**: Access your profile and settings from the top-right corner

The Dashboard displays:

### Sidebar Navigation

The collapsible sidebar provides:

- **Platform Section**: Quick links to Dashboard and main features
- **User Profile**: Your account information and settings
- **Footer Links**: Access to documentation and resources

<Tip>
You can collapse the sidebar by clicking the icon toggle to maximize your workspace.
</Tip>

## Navigation Menu

The main navigation in the sidebar includes:

| Section | Description | Icon |
|---------|-------------|------|
| Dashboard | Overview of your platform | Grid layout icon |
| Series | Manage your series content | Access via content management |
| Media | Upload and manage video files | Media library icon |

## User Menu Options

Click on your profile picture or name in the sidebar footer to access:

    Access all account settings including profile, password, two-factor authentication, and appearance preferences
  </Step>
  <Step title="Log Out">

  </Step>
</Steps>
